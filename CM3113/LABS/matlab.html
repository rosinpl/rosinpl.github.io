Basic Matlab Tutorial

Getting started

Here is a sample session with Matlab. Text in bold is what you type, ordinary text is what the computer "types." You should read this example, then copy it at the computer. 
% matlab

>> a = 1 + 2

a =
     3
>> a^10

ans =
     59049

>> quit

 16 flops.

%
This example adds 1 + 2 and stores the result in variable a. It then computes a to the power 10 and outputs the result. All results are output unless a semi-colon is appending to the command.

More examples of how to use Matlab facilities are given below.

Matrices

To enter the matrix 
             1 2
             3 4
and store it in a variable a, do this: 
       >> a = [ 1 2; 3 4 ]
To redisplay the matrix, just type its name: 
       >> a
To square the matrix a : 
       >> a * a
Now define a matrix b: 
       >> b = [ 1 2; 0 1 ]
and compute the product: 
       >> a * b
and the sum: 
       >> s = a + b
Matrices can sometimes be inverted: 
       >> inv(s)
To check that this is correct, we check that the product of s and its inverse is the identity matrix: 
       >> s * inv(s)

Loops

To loop through a sequence of values try 
       >> for i = 1:4
       >> i,i^2
       >> end

Images

To read in a TIFF image (BMP, HDF, JPEG, PCX, TIFF, XWB formats are supported) do 
    >> i=imread('cameraman.tif');
and display it by 
    >> imshow(i)
If there are insufficient Image Toolbox licences then use the following commands: 
    >> image(i)
    >> colormap(gray)
    >> imagesc(i)
Using imread, the image is stored as a matrix pixel intensities in the range 0 to 255, and are represented by the uint8 datatype. However, many mathematical functions can only be applied to the double class. To convert between uint8 to double (values in the range 0 to 1) do 
    >> i=im2uint8(i);
and from double to uint8. 
    >> i=im2double(i)
Again, if there are insufficient Image Toolbox licences then there are alternatives: 
    >> i = double(i) ./ 255;
and 
    >> i = uint8(i .* 255);
