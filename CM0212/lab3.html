CM0212 - Algorithms II and Data Structures

Lab Sheet #3 - Week 8

Write a Java program that stores a binary tree as an array.

The values can be hardwired into the code to avoid bothering with handling input, and so this part does not involve much coding.

To start off, you should work out how to index a node's parent and children.

Now traverse through the tree in preorder, inorder, and postorder, printing out the node values.
