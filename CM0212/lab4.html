CM0212 - Algorithms II and Data Structures

Lab Sheet #4 - Week 10

Write a Java program to store, remove, and retrieve positive integers using hashing (NOT using the built-in Java Hashtable class).

To keep implementation simple, store entries in an array and handle collisions using linear or double hashing.

You can flag empty/deallocated table entries with negative values (rather than shifting entries to fill the gaps).

To remind yourself how hashing works it would be a good idea to have a look at the demo on my webpage, which includes linear probing and other collision resolution schemes.
