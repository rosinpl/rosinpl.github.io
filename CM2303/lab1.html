Recall the definition of a bag that we covered in the lecture, and the Java code for some of the implementation: 

public class IntArrayBag
{
    private int[] data;
    private int numberItems; 
    
    public IntArrayBag()
    {
        numberItems = 0;
        data = new int[10];
    }
      
    public IntArrayBag(int initialSize)
    {
        if (initialSize < 0)
           throw new IllegalArgumentException
           ("The initialSize is negative: " + initialSize);
        data = new int[initialSize];
        numberItems = 0;
    }
         
    public void ensureSize(int minimumSize)
    {
        int biggerArray[];
       
        if (data.length < minimumSize) {
           biggerArray = new int[minimumSize];
           System.arraycopy(data, 0, biggerArray, 0, numberItems);
           data = biggerArray;
        }
    }
}
